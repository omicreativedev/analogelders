## What Is This

This is the official website of the **Ancient Order of Analog Elders**, an intercollegiate fellowship, scholarly guild consortium, and archival society devoted to the preservation of endangered cognitive disciplines once commonplace among civilized populations prior to the Algorithmic Fragmentation Period.

We safeguard the ancient arts of sustained attention, contextual reasoning, physical media stewardship, and coherent written correspondence. We also invented the internet. This is not up for debate.

The site lives at: **[https://omicreativedev.github.io/analogelders/](https://omicreativedev.github.io/analogelders/)** (for now)

---

## Why We Built This

Because it seemed like the kind of thing that should exist. Because the semicolon deserves a defender. Because someone needed to formally document that BitTorrent is a neutral technology and that the MySpace Top 8 required genuine diplomatic skill.

Also because we had a document titled *"The Ancient Order of Analog Elders: Official Description"* sitting in a folder and it felt wrong not to do something with it.

The Order is tongue-in-cheek. The competencies are real. The frustration with the ellipsis being used aggressively in text messages is deeply sincere.

## Contributing

We genuinely want people to add to this. If you have something funny, historically accurate (to the vibe), or genuinely useful to contribute, please do.

### Ways to help

**Open an Issue** if you notice:
- A bug, a broken layout, a video that no longer works
- Something that should be on the site and is not
- A competency, motto, qualification, or guild we forgot to include
- A typo, a grammatical error, or an incorrect semicolon (the irony would be noted)
- Anything that renders badly on your device or browser

**Open a Discussion** if you want to:
- Propose a new section entirely
- Suggest additional tutorial videos in the spirit of the Order
- Debate the proper rank at which one should learn Gopher Protocol
- Propose a new affiliated guild or subsidiary body
- Question a creative decision and offer a better one

**Submit a Pull Request** if you want to:
- Fix something you found
- Add something you think fits
- Improve responsiveness, accessibility, or performance
- Expand any of the existing sections with more content
- Correct our HTML in a way that earns you bonus consideration for membership

We are not precious about the code. It is a single HTML file. It was written with care but it can always be better. If you see a way to make it better, say so.

### Guidelines (brief, we promise)

- Keep the tone consistent: pompous, affectionate, self-aware
- No actual Masonic symbols (the original brief was clear on this)
- New videos should be genuinely relevant to the Order's mission: analog skills, internet history, privacy, attention, correspondence, the general business of knowing how things work
- New text should be written in the voice of an organization that takes itself very seriously while being fully aware that it is a bit
- Please do not use em dashes in prose. This is a rule now. It is in the Oath.

## Structure

The entire site is one file: `index.html`. No build process, no dependencies, no framework. It uses Google Fonts and YouTube embeds and that is the full extent of its external dependencies.

If you want to run it locally, open it in a browser. That is it. That is the whole process.

```
/
└── index.html       # The entire Order
└── README.md        # You are here
```

## Joining the Order

If you wish to formally petition for membership rather than just contributing to the codebase, the site contains full instructions. The short version: write a proper email, include your case for admission, link to three songs (no MP3 attachments), and allow six weeks for a response. The Council has things going on.

Membership is open to anyone regardless of age. Knowledge and demonstrated skill are the only criteria. Familiarity with HTML earns bonus consideration.

Upon acceptance you will be expected to join the Fediverse if you are not already there. The Elders do not reside on platforms that sell them.

## A Note on the Fax Machine

Several sections of this site reference the Order's legal counsel, described as "a very old fax machine." This is a joke. The Order does not have legal counsel. If you find something on this site that concerns you from a legal standpoint, please open an issue and we will address it like reasonable people who read the whole issue before responding.
